# Jocs en Python

Aquest és un programa senzill amb dos jocs que he fet per practicar Python.

## Que hi ha

El programa té un menú principal amb dos jocs:

1. Pedra, paper o tisores
2. Endevinar números

## Com funciona

Executa el fitxer Main.py i tria l'opció que vulguis.

### Pedra, paper o tisores

Primer tries si vols jugar al millor de 3 o al millor de 5. Després escrius pedra, paper o tisores i el robot tria una opció aleatòria. Guanya qui arribi primer a 2 victories (al millor de 3) o a 3 victories (al millor de 5).

### Endevinar números

El programa pensa un número entre 1 i 100 i has d'endevinar-lo. Et dirà si el teu número és massa alt o massa baix fins que l'encertis.

## Fitxers

- Main.py: el menú principal
- jocs.py: els dos jocs
- robot.py: classe del robot (encara no s'utilitza)

## Com executar

python Main.py
